
# 🧮 Advanced Scientific Calculator (Web + Python)

🚀 A modern and interactive **multi-functional calculator** built using Python and Web technologies.  
This project includes CLI, GUI, and a **feature-rich web calculator with scientific operations and history tracking**.

---

## 📌 Overview
This calculator project is designed to provide both **basic and advanced mathematical operations** in multiple formats:
- Command-Line Interface (CLI)
- Graphical User Interface (GUI)
- Web-based Scientific Calculator

The web version includes a **modern UI, glowing effects, and calculation history panel**, making it user-friendly and visually appealing.

---

## ✨ Features

### 🔹 Web Calculator (Main Highlight)
- 🧠 Supports **scientific functions**:
  - sin, cos, log, π, √
- ➕ Basic operations:
  - Addition, Subtraction, Multiplication, Division
- 📜 **History panel**:
  - Displays previous calculations
- 🎨 Modern UI:
  - Glassmorphism design
  - Smooth layout with soft colors
- 🌙 Dark mode toggle (top right)
- ⚡ Real-time calculation

---

### 🔹 CLI Calculator (Python)
- Advanced operations:
  - Power, Square Root, Modulus
- Input validation:
  - Prevents invalid characters
- Error handling:
  - Division by zero, invalid input

---

### 🔹 GUI Calculator (Python - Tkinter)
- Button-based calculator interface
- Simple and easy to use

---
